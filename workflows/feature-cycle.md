# Workflow: Full Feature Delivery Cycle
Command: `/feature-cycle`

## Phase 1: Discovery (Product & Design)
1. **Product Manager** defines the PRD.
2. **Business Analyst** creates User Stories & Acceptance Criteria.
3. **UI/UX Designer** drafts the interface specs.

## Phase 2: Design (Architecture)
4. **System Architect** defines the data flow and component hierarchy.
5. **Tech Lead** reviews and approves the technical strategy.

## Phase 3: Breakdown (Management)
6. **Task Planner** explodes the feature into granular sub-tasks.
7. **Engineering Manager** assigns tasks to specific Agents (BE, FE, DB).

## Phase 4: Execution (Development)
8. **API Engineer** sets the contract.
9. **Backend/Frontend/Database Engineers** implement the code in parallel.
10. **Refactoring Specialist** & **Code Reviewer** perform real-time audits.

## Phase 5: Verification (QA)
11. **QA Engineer** validates manual flows.
12. **Test Automation** runs E2E and Integration suites.
13. **Bug Investigator** fixes blockers found during testing.

## Phase 6: Closure (Documentation & Release)
14. **Documentation Writer** updates README and API docs.
15. **DevOps Engineer** triggers the release via **Release Master** skill.
