# Frontend Master Skill

## UI Conventions
- Mobile-first responsive design using Tailwind utility classes.
- Use `lucide-react` for icons and `shadcn/ui` for base components.

## Performance Check
- Avoid unnecessary re-renders using `useMemo` and `useCallback`.
- Implement lazy loading for heavy components.
- Ensure 100% accessibility (ARIA labels, keyboard navigation).
